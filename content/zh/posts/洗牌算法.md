## 暴力洗牌

n个元素共有n!个排列，暴力枚举每种情况，从中获取的一种作为洗牌结果。O(n!)复杂度。

## Knuth洗牌算法

```c#
public static void Shuffle<T>(IList<T> items)
{
    var rn = new Random((int) DateTime.Now.Ticks);
    for (var i = items.Count - 1; i >= 0; i--)
    {
        Swap(items, i, rn.Next(i + 1));
    }
}
```
