Unity的Android开发环境配置

Android环境配置

  1. 添加 Android 开发相关组件

  2. 启用 USB 调试

将手机设置为开发者模式，具体机型可能有所不同，但通常都是连续点击设置里的 Build number 多次即可进入开发者模式。之后在开发者选项里启用 USB 调试。

其他

使用自定义的 SDK 路径

可在 Preferences > External tools 里设置自定义的 SDK等路径

调整 Android SDK Api Level

Edit > Project Settings > Player 下可设置此选项
